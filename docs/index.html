Diceware en Español

El Diceware es un método para generar contraseñas seguras usando palabras elegidas al azar.

  • Se usan dados físicos o números aleatorios seguros.
  • Cada combinación apunta a una palabra.
  • La lista tiene 7776 palabras.
  • Se repite el proceso para crear una frase de paso.

Cómo usar esta herramienta:

  1. Elige el separador: Selecciona el carácter que separará las palabras (guión, punto, etc.)
  2. Ajusta el número de palabras: Usa el slider para elegir entre 4 y 8 palabras. Más palabras = contraseña más fuerte.
  3. Genera tu contraseña: Haz clic en "Generar" para crear una nueva frase de paso.
  4. Copia la contraseña: Haz clic en "Copiar" para copiarla al portapapeles.
  5. Memorización: Intenta memorizar la frase - son palabras comunes en español, más fáciles de recordar que caracteres aleatorios.

💡 Consejo: Una frase de 5 palabras es segura para la mayoría de usos. 6 palabras es muy segura incluso para datos críticos.

¿Qué es la entropía?

La entropía mide qué tan difícil es adivinar una contraseña. Se expresa en bits y depende del número de combinaciones posibles.

En este generador, cada palabra se elige de una lista de 7776 palabras. Cada palabra aporta aproximadamente 12.9 bits de entropía.

  • 4 palabras ≈ 52 bits → bueno
  • 5 palabras ≈ 65 bits → fuerte
  • 6 palabras ≈ 78 bits → muy fuerte

Agregar números o mayúsculas ayuda a cumplir requisitos de algunas páginas, pero la mayor seguridad viene de usar más palabras.

💡 Recomendación: usa al menos 5 palabras para contraseñas importantes.
